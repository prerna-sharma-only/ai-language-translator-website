# 🌍 AI Voice Translator

An advanced **AI-powered real-time voice & text translation web application** that allows users to speak or type in one language and instantly receive translated output along with natural speech synthesis.

This project delivers a **premium app-like multilingual communication experience** by combining modern UI design, browser speech APIs, and lightweight backend integration.

---

## 🚀 Key Features

✨ Real-time voice-to-text translation
✨ Automatic live typing translation
✨ Multi-language support (20+ global languages)
✨ Natural AI speech output (Text-to-Speech)
✨ Smart language swap & auto detection logic
✨ Chat-style translation conversation interface
✨ AI typing animation & thinking indicators
✨ Siri-style live microphone sound wave animation
✨ Translation history dashboard
✨ Premium glassmorphism UI with cinematic background
✨ Export translation chat as image (branding watermark included)
✨ Smooth glow cursor & modern interactive animations

---

## 🧠 Technologies Used

* **Node.js**
* **Express.js**
* **translate-google API**
* **Web Speech API** (Speech Recognition + Speech Synthesis)
* **Vanilla JavaScript (ES6)**
* **HTML5**
* **Advanced CSS3** (Glass UI, Gradients, Animations, Responsive Design)
* **html2canvas** (Export chat as image feature)

## 🛠️ Installation & Run Locally

```bash
git clone https://github.com/prerna-sharma-only/ai-voice-translator.git
cd ai-voice-translator
npm install
node server.js
```

Now open your browser and visit:

http://localhost:5000
